# 향기짱 시간표

애견미용실 주간 예약 관리를 위한 iOS 앱입니다. SwiftUI와 SwiftData로 만들어졌습니다.

## 주요 기능

- **주간 시간표**: 화~토 기준 주간 그리드에서 예약 현황을 한눈에 확인하고, 좌우로 스와이프해 주를 이동합니다. 빈 칸을 탭하면 예약을 추가하고, 예약을 탭하면 수정할 수 있습니다.
- **예약 등록/수정**: 서비스 종류(전체, 가위컷, 가위컷(비숑), 부목얼, 부목얼(비숑), 목욕, 위생)에 따라 기본 소요 시간이 자동으로 계산되며, 필요 시 시간을 조정할 수 있습니다. 영업시간(10:00~20:00), 영업일(화~토), 기존 예약과의 중복, 휴무일 여부를 검사해 예약 가능 여부를 판단합니다.
- **회원(반려견) 관리**: 반려견 이름, 견종 메모, 연락처를 등록하고 이름/견종/전화번호로 검색할 수 있습니다.
- **공휴일 표시**: EventKit으로 기기의 대한민국 공휴일 캘린더를 읽어와 시간표에 공휴일과 설날/추석 라벨을 표시합니다. 최초 실행 시 캘린더 접근 권한이 필요합니다.
- **휴무일 지정**: 특정 날짜를 휴무로 지정해 해당 날짜의 예약을 막을 수 있습니다.
- **대기 메모**: 자유 형식의 메모를 기기에 저장해두는 대기 탭을 제공합니다.

앱은 시간표 / 회원 / 대기 세 개의 탭으로 구성되어 있습니다.

## 기술 스택

- SwiftUI
- SwiftData (로컬 데이터 저장: 예약, 반려견, 휴무일)
- EventKit (공휴일 연동)
- Swift 5.0, iOS 26.4+

## 프로젝트 구조

```
TimeTable/
├── Models/        # SwiftData 모델 (Appointment, Dog, DayOff)
├── Views/         # 화면 (시간표 그리드, 예약 등록/수정, 회원 목록, 대기 메모 등)
├── ViewModels/     # 화면 상태/로직 (주간 이동, 공휴일 로딩)
└── Utilities/      # 보조 로직 (시간 슬롯 계산, 회원 마이그레이션 등)
```

## 실행 방법

1. `TimeTable.xcodeproj`를 Xcode에서 엽니다.
2. 시뮬레이터 또는 실제 기기를 선택합니다.
3. ⌘R로 빌드 및 실행합니다.
4. 공휴일 표시 기능을 사용하려면 최초 실행 시 캘린더 접근 권한을 허용해야 합니다.

## 테스트

Xcode에서 ⌘U를 실행하면 `TimeTableTests`(유닛 테스트)와 `TimeTableUITests`(UI 테스트) 타깃이 실행됩니다.
